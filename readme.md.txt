# Employee Knowledge Assistant

## Overview

A hands-on project to learn Retrieval-Augmented Generation (RAG) by building an AI assistant capable of answering employee policy and process-related questions.

## Current Version

✅ Version 1 - Static MVP

This version demonstrates:

- Streamlit UI
- User question input
- Static responses
- Source references
